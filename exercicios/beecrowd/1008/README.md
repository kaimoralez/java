# 💰 Exercício 1008 - Salário

## 📌 Descrição
Este exercício consiste em calcular o salário de um funcionário com base nas horas trabalhadas e no valor recebido por hora.

## 🚀 Conceitos utilizados
- Entrada de dados com Scanner
- Variáveis do tipo int e double
- Operações matemáticas
- Saída formatada com printf

## 🧠 Lógica da solução
1. Ler o número do funcionário
2. Ler a quantidade de horas trabalhadas
3. Ler o valor recebido por hora
4. Calcular o salário total (horas * valor por hora)
5. Exibir o número do funcionário
6. Exibir o salário formatado com duas casas decimais

## 💻 Exemplo
Entrada:
25
100
5.50

Saída:
NUMBER = 25
SALARY = U$ 550.00

## 📚 Aprendizados
- Trabalhar com múltiplos tipos de dados
- Uso do printf para formatar valores monetários
- Organização da saída em múltiplas linhas
