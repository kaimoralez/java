# 🧮 Exercício 1007 - Diferença

## 📌 Descrição
Este exercício consiste em ler quatro valores inteiros (A, B, C e D) e calcular a diferença entre o produto de A e B e o produto de C e D.

## 🚀 Conceitos utilizados
- Entrada de dados com Scanner
- Variáveis do tipo int
- Operações matemáticas (multiplicação e subtração)
- Saída de dados

## 🧠 Lógica da solução
1. Ler quatro valores inteiros
2. Calcular o produto de A * B
3. Calcular o produto de C * D
4. Subtrair os dois resultados
5. Exibir o valor final

## 💻 Exemplo
Entrada:
5
6
7
8

Saída:
DIFERENCA = -26

## 📚 Aprendizados
- Importância da ordem das operações
- Diferença entre caracteres com e sem acento
- Atenção à formatação exata exigida pelo Beecrowd
