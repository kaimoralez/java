# 🧮 Exercício 1006 - Média 2

## 📌 Descrição
Este exercício consiste em calcular a média ponderada de três notas. Cada nota possui um peso diferente:
- A = peso 2
- B = peso 3
- C = peso 5

## 🚀 Conceitos utilizados
- Entrada de dados com Scanner
- Uso de variáveis do tipo double
- Operações matemáticas
- Média ponderada
- Saída formatada com printf

## 🧠 Lógica da solução
1. Ler três valores do tipo double
2. Multiplicar cada valor pelo seu respectivo peso
3. Somar os resultados
4. Dividir pela soma dos pesos (10)
5. Exibir o resultado com uma casa decimal

## 💻 Exemplo
Entrada:
5.0
6.0
7.0

Saída:
MEDIA = 6.3

## 📚 Aprendizados
- Como calcular média ponderada
- Diferença entre média simples e ponderada
- Uso do printf para controlar casas decimais
- Importância da formatação exata no Beecrowd
